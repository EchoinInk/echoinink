ECHO IN INK — CARD SYSTEM BLUEPRINT
Unified Component Architecture for Transformation, Systems, and Worlds
---
1. Purpose of This System
Create a cohesive, atmospheric, premium card architecture that supports all Echo in Ink experiences across Studio, Systems, and LUMO.
Cards are not UI elements.
They are atmospheric containers — portals into worlds, systems, and transformations.
This blueprint defines the three canonical card types, their shared foundations, and their emotional distinctions.
---
2. Canonical Card Types
There are only three true card archetypes in the Echo in Ink ecosystem:
1. Transformation Cards
Brand Identity / Narrative Architecture
Emotional, spacious, editorial.
2. Systems Cards
Identity Canvas / GPT Systems / UI Kit
Infrastructural, archival, intelligent.
3. Featured World Cards
LUMO / World Features
Cinematic, immersive, narrative.
All other cards are variations of these three.
---
3. Core Card Foundation
All cards share the same atmospheric base.
---
3.1 Background Layer
background:
linear-gradient(
  180deg,
  rgba(10,12,30,0.72),
  rgba(5,6,18,0.92)
);

Deep, atmospheric gradient
Never pure black
Creates world‑depth and visual quiet
---
3.2 Border
--border-card 1px solid

Whisper‑thin
Almost invisible
Light catching glass, not UI chrome
---
3.3 Radius
--radius-card: 28px;

Avoids tech‑startup sharpness
Avoids SaaS pill‑roundness
Feels crafted, intentional, premium
---
3.4 Backdrop Blur
backdrop-filter: blur(24px);

Atmospheric depth
Softens background noise
Creates spatial layering
---
3.5 Shadow
box-shadow:
0 0 60px rgba(90,60,255,0.08);

Ambient glow
No hard shadows
No UI‑style elevation

6. Density & Luxury Principles
Low Density = Premium
Echo in Ink’s strongest designs succeed because they contain:
minimal text
generous breathing room
restrained hierarchy
slow, emotional pacing
Luxury systems remove, not add.
---
7. Visual System Rules
1. No Pure Black
Use:
--ei-deep-midnight

2. Glow Is Sacred
Use glow only for:
focal visuals
monograms
system nodes
CTA anchors
If everything glows, nothing feels sacred.
3. Borders Should Almost Disappear
Barely visible.
Glass‑edge shimmer.
4. Typography Does the Heavy Lifting
Atmosphere comes from type, not ornament.
5. Spacing Creates Luxury
Increase:
top padding
breathing room
paragraph spacing
More than you think.
