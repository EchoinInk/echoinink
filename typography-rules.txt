ECHO IN INK — Typography System
Font Roles
Fraunces
Use for emotional display moments only:
hero headlines
manifesto lines
poetic section titles
cinematic statements
editorial emphasis
Fraunces carries the emotional gravity of the brand.
It should feel atmospheric, restrained, and intentional.
Neue Montreal
Use for everything practical:
body copy
navigation
buttons
cards
descriptions
forms
UI systems
Neue Montreal provides clarity and modern precision without feeling sterile.
Geist Mono
Use sparingly for system language:
categories
metadata
labels
tags
technical cues
architectural references
Geist Mono introduces subtle system intelligence and structured rhythm.
Desktop Typography Rules
Hero Headline
Font: Fraunces
Size: 72–96px
Line-height: 0.95–1.05
Letter spacing: -0.04em
Weight: 400–500
Used for:
hero sections
cinematic opening statements
emotional positioning moments
Large Section Statement
Font: Faunces
Size: 48–64px
Line-height: 1.05
Letter spacing: -0.035em
Weight: 400
Used for:
philosophy statements
manifesto blocks
immersive editorial sections
Section Heading
Font: Faunces or Neue Montreal
Size: 32–44px
Line-height: 1.1
Letter spacing: -0.025em
Used for:
major content sections
case studies
system overviews
Body Copy
Font: Neue Montreal
Size: 17–19px
Line-height: 1.55–1.7
Letter spacing: -0.005em
Weight: 400
Body copy should feel:
calm
breathable
intelligent
emotionally restrained
Small Body / Card Copy
Font: Neue Montreal
Size: 14–16px
Line-height: 1.5–1.65
Weight: 400
Used for:
cards
descriptions
supporting UI content
captions
Navigation
Font: Neue Montreal
Size: 13–14px
Line-height: 1
Letter spacing: 0.02em
Weight: 400–500
Navigation should feel:
minimal
architectural
effortless
spacious
Buttons
Font: Neue Montreal
Size: 14–15px
Letter spacing: 0.01em
Weight: 500
Buttons should feel:
calm
premium
understated
intentional
Avoid oversized CTA styling.
System Labels
Font: Geist Mono
Size: 10–12px
Transform: Uppercase
Letter spacing: 0.12–0.18em
Weight: 400–500
Used for:
section labels
metadata
interface systems
atmospheric annotations
Mobile Typography Rules
Hero Headline
Font: Faunces
Size: 44–56px
Line-height: 0.95–1.05
Letter spacing: -0.035em
Section Statement
Font: Faunces
Size: 34–42px
Line-height: 1.05–1.1
Section Heading
Font: Faunces / Neue Montreal
Size: 26–32px
Line-height: 1.1
Body Copy
Font: Neue Montreal
Size: 16–17px
Line-height: 1.55–1.65
Labels
Font: Geist Mono
Size: 10–11px
Letter spacing: 0.12em
Color + Contrast Rules
Primary Text
Use:
#F6F2FF
Body Copy Opacity
Use:
75–85% opacity
This creates softness without losing readability.
Secondary Copy
Use:
58–68% opacity
For:
metadata
supporting descriptions
atmospheric hierarchy
Minimum Readability
Never place body copy below:
16px
Accent Usage
Use magenta or violet only for:
emphasis
interaction states
subtle highlights
Never use bright gradients for full paragraphs.
Hierarchy Rule
Each section should use no more than:
1 display style
1 body style
1 label style
This restraint is what keeps Echo in Ink feeling:
cinematic
premium
emotionally intelligent
calm
spacious
Best Pairing Structure
Faunces = soul
Emotional gravity. Editorial atmosphere. Human resonance.
Neue Montreal = clarity
Precision. Structure. Readability. Modern restraint.
Geist Mono = system intelligence
Architecture. Metadata. Quiet technical rhythm.
Typography Philosophy
Echo in Ink typography should feel:
authored, not templated
cinematic, not corporate
emotional, not decorative
spacious, not dense
restrained, not performative
Typography is not decoration.
It is atmosphere translated into language.
